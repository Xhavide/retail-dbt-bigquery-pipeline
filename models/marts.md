{% docs customer_behavior_summary %}
A specialized dimensional mart table that aggregates customer demographic segments (Age Cohorts and Gender) against product pricing tiers. It is materialized as a physical table to pre-compute financial KPIs and optimize BI dashboard query speeds.
{% enddocs %}


{% docs retail_gross_revenue %}
The absolute cumulative sales revenue generated by a specific retail cohort, calculated as `quantity_purchased * price_per_unit`. This metric is strictly monitored to ensure values never drop below zero.
{% enddocs %}
